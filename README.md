### AES-GCM Encryption Tool
<pre>Usage of aescrypt:
  -d    Decrypt instead Encrypt.
  -f string
        Target file.
  -i int
        Iterations. (for PBKDF2) (default 1024)
  -k string
        256-bit key to Encrypt/Decrypt.
  -p string
        PBKDF2.
  -r    Generate random 256-bit cryptographic key.
  -s string
        Salt. (for PBKDF2)</pre>
